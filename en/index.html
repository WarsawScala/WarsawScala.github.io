WarszawScala

Info

WarszawScala is Warsaw user group of Scala programming language. The main purpose of the group is to connect local community of Scala programmers. We learn together and exchange opinions about fascinating world of functional programming ;-). We coordinate meetings via Meetup group, we discuss online using Google group.
